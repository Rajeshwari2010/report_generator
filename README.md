# 📊 Report Generator System

A full-stack web application designed to simplify and automate the creation of structured reports. This system allows users to create, edit, and export reports with dynamic content, making it useful for academic and departmental use.

---

## 🚀 Features
- 📝 Create detailed reports with sections like objectives, outcomes, feedback, and summary  
- ✏️ Edit existing reports with pre-filled data  
- 📄 Generate downloadable PDF reports  
- 🔐 Secure authentication system  
- 📊 Efficient data management using backend APIs  
- ⚡ Responsive and user-friendly interface  

---

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Tools:** jsPDF, REST APIs

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/report_generator.git
cd report_generator
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run backend
```bash
cd server
npm install
npm start
```

### 4. Run frontend
```bash
cd client
npm install
npm start
```
